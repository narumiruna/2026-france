# 餐廳偏好（已定稿）

本文件定義使用者的食物偏好，供研究、篩選與地圖決策直接使用。

## 1. 全域硬性規則（Hard Rules）
- 不喝酒是最重要事項。
- 可吃 Bistro，但不得以酒精消費作為用餐前提。
- 每天都要有高品質咖啡選項（早餐或下午時段至少一個）。

## 2. 全域偏好方向
- 主策略：穩定不踩雷。
- 主要價格帶：`€€` 為主。
- 預約容忍度：可提前 `3-7` 天預約。
- 排隊容忍度：超過 `20` 分鐘就切換備選。

## 3. 城市細分食物偏好（依行程順序）
### 3.1 Avignon
- 高優先：
  - 普羅旺斯燉菜（核心目標）。
- 中優先：
  - 海鮮。
  - 甜點/烘焙。
  - 咖啡館。
- 條件規則：
  - Bistro 可接受，但仍套用「不喝酒」硬規則。

### 3.2 Nice
- 已選料理與優先順序：
  - `1) Daube niçoise`
  - `2) Pissaladière`
  - `3) Socca`
- 執行重點：
  - 優先吃到前兩項，再用 Socca 補位。

### 3.3 Mont Saint-Michel
- 已選料理：
  - `Omelette de la Mère Poulard`（主力目標）。
- 執行重點：
  - 停留時間短，先確保主力店可執行，再考慮備選。

### 3.4 Paris
- 已選項目：
  - `Steak frites`
  - `Bistro 經典主菜`
  - `Croissant / Viennoiserie`
  - `甜點店`
  - `咖啡館`
  - `現代法式套餐`
- 城市主軸：
  - 經典 Bistro 正餐優先。
- 條件規則：
  - 即使在 Bistro 或套餐店，仍維持不喝酒。

## 4. 選店判定規則
- 符合硬性規則且命中城市高優先料理：列為主力候選。
- 符合硬性規則但只命中中優先料理：列為備選。
- 任何與不喝酒規則衝突的體驗（例如強綁酒精 pairing）：降級或排除。
